Vec2f VGAIL::Boid::wander ( f32 circleDistance,
f32 circleRadius,
f32 displacementRange,
f32 maxAcceleration )
inline

Performs the wander steering behavior.

This implementation follows Craig Reynold's proposal, uses a circle defined in front of the character from which the steering force is calculated. Every frame, a random point is chosen from the outline of this circle which will the new direction the character will need to steer towards. In order to limit flickering, a displacementRange is needed such that it limits the interval from which this random point is chosen.

Parameters
circleDistanceThe distance from this Boid object to the circle.
circleRadiusThe radius of the circle.
displaceRangeThe range within which the random point is chosen.
maxAccelerationThe maximum rate at which the velocity can change per unit of time.
Returns
Vec2f The steering force used to wander.
