DecisionNode & VGAIL::DecisionNode::addChild ( Args &&... args)
inline

Creates a new DecisionNode object and adds it to the list of child nodes of this DecisionNode object.

This is a templated method, meaning any custom class can be added as a child node as long as it inherits from DecisionNode.
Any number of variables can be passed, but if there is a mismatch of variables with the T constructor then there will be a compile time error.
