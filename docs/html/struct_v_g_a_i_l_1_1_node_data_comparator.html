Compares two NodeData objects based on their f() value.

Used in the A* algorithm when managing the priority queue that contains all nodes to be looked at during the algorithm.
Whenever a NodeData object is added to the queue, it checks where to put it depending on its f() value by comparing it to the other objects in the list.
