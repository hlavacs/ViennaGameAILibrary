template<class T , typename... Args>
DecisionNode & VGAIL::DecisionTree::createRoot ( Args &&... args)
inline

Creates and returns the root of the decision tree.

This is a templated method, meaning any custom class can be used as a root node as it inherits from DecisionNode.
Any number of variables can be passed, but if there is a mismatch of variables with the T constructor then there will be a compile time error.
