◆ findPreprocessedPath()

std::vector< Vec2ui > VGAIL::NavMesh::findPreprocessedPath ( Vec2ui start,
Vec2ui target )
inline

Finds the stored path between two nodes.

After geometric preprocessing, the shortest paths between each node to each region is stored at m_adjList[nodeIndex][regionID].

  • If the nodes are in the same region, it simply calls the A* algorithm to find the shortest path between them.
  • If they are in separate regions, it first retrieves the shortest path from the start node to the region in which the target node exists. If the newly found path's end node is the target, the path is returned. Otherwise, A* will run to find the path from this end node to the target node and return it, if found.
Parameters
startThe position of the start node.
targetThe position of the target node.
Returns
std::vector<Vec2ui> The shortest found path.

◆ preprocess()

void VGAIL::NavMesh::preprocess ( bool multithreading = false,
ui32 numThreads = 4 )
inline

Performs geometric preprocessing with optional multithreading.

Parameters
multithreadingSet to true to enable multithreading, false otherwise.
numThreadsThe number of threads to use if multithreading is enabled.

a) Multithreaded
Each thread receives a part of the already defined regions and is assigned the preprocessWorker() method.

b) Single-threaded
For each node, calculate the shortest path to each region and store it at position m_adjList[nodeIndex][regionID], where nodeIndex is the index of the node within m_nodes and regionID is the ID of the region it found a path to. m_adjList will then be used to retrieve shortest path within regions.
