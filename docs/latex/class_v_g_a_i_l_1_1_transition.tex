\doxysubsection{Detailed Description}
Custom class that manages the behavior of transitions between states. 

Used in state machines. ~\newline
These transitions lead from one \doxylink{class_v_g_a_i_l_1_1_state}{State} to another (the {\ttfamily target} state) based on a set of given conditions. If the conditions are met, the {\ttfamily target} state becomes active.
