Finds the stored path between two nodes. 

After geometric preprocessing, the shortest paths between each node to each region is stored at {\ttfamily m\+\_\+adj\+List\mbox{[}node\+Index\mbox{]}\mbox{[}region\+ID\mbox{]}}. ~\newline

\begin{DoxyItemize}
\item If the nodes are in the same region, it simply calls the A\texorpdfstring{$\ast$}{*} algorithm to find the shortest path between them. ~\newline

\item If they are in separate regions, it first retrieves the shortest path from the start node to the region in which the target node exists. If the newly found path\textquotesingle{}s end node is the target, the path is returned. Otherwise, A\texorpdfstring{$\ast$}{*} will run to find the path from this end node to the target node and return it, if found.
\end{DoxyItemize}
