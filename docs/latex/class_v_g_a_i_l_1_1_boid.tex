Performs the wander steering behavior. 

This implementation follows Craig Reynold\textquotesingle{}s proposal, uses a circle defined in front of the character from which the steering force is calculated. Every frame, a random point is chosen from the outline of this circle which will the new direction the character will need to steer towards. In order to limit flickering, a {\ttfamily displacement\+Range} is needed such that it limits the interval from which this random point is chosen.
